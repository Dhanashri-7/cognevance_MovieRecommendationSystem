# cognevance_MovieRecommendationSystem

## Project Description
An AI-based recommendation system built to suggest movies based on user interests and ratings.

## Technologies Used
* **Language**: Python
* **Libraries**: Pandas, NumPy, Matplotlib, Seaborn

## Project Workflow
1. **Data Collection**: Sourced movie ratings data.
2. **Cleaning**: Removed null values and duplicates.
3. **Analysis**: Explored movie categories and user rating trends.
4. **Modeling**: Applied Collaborative Filtering to generate recommendations.
5. **Output**: Displayed recommended movies and visualizations.

## Repository Structure
* `/data`: Contains the movie and ratings datasets.
* `/src`: Python source code (Notebook/Script).
* `/screenshots`: Visualizations and system outputs.
* `Report.md`: Detailed project report.
